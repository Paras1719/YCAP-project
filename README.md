*Spring Boot Login Application (JDBC + MySQL)*

A simple Spring Boot login application using Thymeleaf, JDBC, and MySQL.
This project demonstrates form submission, database authentication, and MVC structure.

#Features

Login UI using Thymeleaf

Validates username and password from MySQL

JDBC Template for database queries

Error and success page handling

Fully configurable using application.properties
